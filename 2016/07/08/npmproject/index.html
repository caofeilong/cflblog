将项目发布到npm社区

Jul 8, 2016 | Hits

Q3 KPI fengui npm项目创建记录

步骤

  1. 安装node，npm
  2. 创建项目目录
  3. npm init
  4. 编写自己的项目
  5. 发布

发布

创建用户 npm adduser

登录用户 npm login
创建完用户之后，执行npm publish --tag 0.1.0 发布到npm社区

执行npm install [项目名] 安装依赖

执行npm unpublist[<@scope>/]<pkg>[@<version>] 卸载包



参考地址：https://docs.npmjs.com/cli
