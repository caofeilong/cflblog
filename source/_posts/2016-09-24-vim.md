## vim 常用的指令

```
    i    ->  insert   按esc 进入  normal 模式
    x    ->  删除当前光标所在的一个字符
    :wq  ->  保存并退出
    dd   ->  删除当前行
    :q!   ->  退出不保存
    :W   ->  保存不退出
    //光标移动
    o    ->  在当前行后插入一个新行
    O    ->  在前一行插入
    cw   ->  删除光标之后的信息
    $    ->  移至行末
    ^    ->  移至行首

    p  粘贴
    yy  复制当前行
    u   撤销
    ctl+r   恢复
    :e   打开一个文件
    :bn  下一个文件
    :bp   上衣个文件

```
